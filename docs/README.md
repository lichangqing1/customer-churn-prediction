## Important implementation notes

- `mart_segment_churn` does not contain `churned_customers`. Use `customers × churn_rate` if an estimated churned-customer count is needed in Power BI.
- `mart_revenue_at_risk` and `mart_retention_actions` are based on the 1,409-customer scored artifact in `results/risk_table_with_recommendations.csv`.
- The full warehouse contains 7,043 customers from `data/raw/WA_Fn-UseC_-Telco-Customer-Churn.csv`.
- Runtime artifacts such as SQLite databases and Power BI CSV exports are generated by scripts and intentionally ignored by Git.
