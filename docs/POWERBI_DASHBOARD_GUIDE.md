# Power BI Dashboard Guide

This guide explains how to build the Power BI dashboard from the actual mart CSV files generated by the project.

---

## 1. Generate Power BI data files

Run the following commands from the project root:

```bash
python scripts/build_churn_warehouse.py
python scripts/export_powerbi_data.py
```

The Power BI files are created in:

```text
results/powerbi_exports/
```

Files:

```text
mart_churn_overview.csv
mart_segment_churn.csv
mart_revenue_at_risk.csv
mart_retention_actions.csv
mart_data_quality_status.csv
```

These files are ignored by Git because they are reproducible.

---

## 2. Import files into Power BI

In Power BI Desktop:

```text
Home → Get Data → Text/CSV
```

Import all five CSV files.

Recommended table names:

```text
mart_churn_overview
mart_segment_churn
mart_revenue_at_risk
mart_retention_actions
mart_data_quality_status
```

No relationships are required for the basic dashboard because each mart is already dashboard-ready.

## 4. Core DAX measures

Create a measure table:

```DAX
Measures = { 0 }
```

### Executive overview

```DAX
Total Customers =
SUM(mart_churn_overview[total_customers])
```

```DAX
Overall Churn Rate =
AVERAGE(mart_churn_overview[overall_churn_rate])
```

```DAX
Total Monthly Revenue =
SUM(mart_churn_overview[total_monthly_revenue])
```

```DAX
Average Monthly Charge =
AVERAGE(mart_churn_overview[avg_monthly_charges])
```

### Churned customer analysis

```DAX
Segment Customers =
SUM(mart_segment_churn[customers])
```

```DAX
Estimated Churned Customers =
SUMX(
    mart_segment_churn,
    mart_segment_churn[customers] * mart_segment_churn[churn_rate]
)
```

```DAX
Weighted Churn Rate =
DIVIDE(
    [Estimated Churned Customers],
    [Segment Customers]
)
```

```DAX
Highest Segment Churn Rate =
MAX(mart_segment_churn[churn_rate])
```

### Revenue at risk

```DAX
Total Revenue at Risk =
SUM(mart_revenue_at_risk[revenue_at_risk])
```

```DAX
Customers at Risk =
DISTINCTCOUNT(mart_revenue_at_risk[customer_id])
```

```DAX
Average Churn Probability =
AVERAGE(mart_revenue_at_risk[churn_probability])
```

```DAX
Average Monthly Charge Risk =
AVERAGE(mart_revenue_at_risk[monthly_charges])
```

### Retention action queue

```DAX
Customers in Action Queue =
DISTINCTCOUNT(mart_retention_actions[customer_id])
```

```DAX
P1 Customers =
CALCULATE(
    DISTINCTCOUNT(mart_retention_actions[customer_id]),
    mart_retention_actions[retention_priority] = "P1"
)
```

```DAX
High Risk Customers =
CALCULATE(
    DISTINCTCOUNT(mart_retention_actions[customer_id]),
    mart_retention_actions[risk_level] = "High Risk"
)
```

```DAX
Estimated Saved Revenue =
SUM(mart_retention_actions[estimated_saved_revenue])
```

```DAX
Action Queue Revenue at Risk =
SUM(mart_retention_actions[revenue_at_risk])
```

### Data quality monitor

```DAX
Raw Rows =
SUM(mart_data_quality_status[raw_rows])
```

```DAX
Silver Rows =
SUM(mart_data_quality_status[silver_rows])
```

```DAX
Fact Rows =
SUM(mart_data_quality_status[fact_rows])
```

```DAX
Reconciliation Variance =
SUM(mart_data_quality_status[reconciliation_variance])
```

```DAX
Failed Quality Checks =
SUM(mart_data_quality_status[failed_quality_checks])
```

```DAX
Data Quality Status =
IF(
    [Failed Quality Checks] = 0
        && [Reconciliation Variance] = 0,
    "Passed",
    "Failed"
)
```

## 10. Formatting tips

### Show exact numbers instead of `7K`

For cards or chart labels:

```text
Format visual → Display units → None
Decimal places → 0
```

If the option is hidden in a new Power BI card visual, use a text-formatted DAX measure:

```DAX
Raw Rows Display =
FORMAT([Raw Rows], "#,##0")
```

### Conditional formatting

For tables:

- `churn_rate`: background color rules
- `churn_probability`: background color rules
- `revenue_at_risk`: data bars
- `estimated_saved_revenue`: data bars
- pass/fail columns: green for 1, red for 0
